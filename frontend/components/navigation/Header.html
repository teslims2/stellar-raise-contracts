Header Component Demo

Resize the browser window to observe the responsive behaviour:

  • < 768 px (mobile) — Site header visible at top, BottomNav visible at bottom.
  • ≥ 768 px (tablet) — Site header hidden, Sidebar visible on the left.
  • ≥ 1024 px (desktop) — Sidebar widens to 280 px.

Scroll down to trigger the .site-header--scrolled modifier, which elevates the header shadow from --shadow-sm to --shadow-md.
